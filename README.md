# OsiCal - 推し活スケジュール管理アプリ

**OsiCal** は、推し（グループ・個人）に関するイベントを一元管理できる、個人向け推し活スケジュール管理アプリです。  
イベント・ライブ・配信・応募締切など、カテゴリーに応じて異なる項目を柔軟に扱えるよう設計しています。

トップページにはカレンダー（FullCalendar）を採用し、予定を視覚的に確認できる UI を実現しています。

---

## 🎯 概要（Overview）

推し活では、イベント、ライブ応募、配信スケジュールなど管理すべき予定が多くなりがちです。  
OsiCal は「推しに関する予定をすべて可視化し整理できるツール」を目指して開発しました。

- 推し（グループ・個人）の管理  
- イベントのカテゴリー別登録・編集  
- カレンダーによる視覚的な予定管理  
- 推し別イベント一覧  
- カテゴリー選択によるフォーム自動切替  
- セッションによる入力保持・カレンダー状態保持

## ✨ 主な機能（Features）

### 🔹 1. カレンダー表示（FullCalendar）
- カレンダーにすべてのイベントを表示  
- イベントクリックで詳細画面に遷移  
- カレンダーの表示月をセッションで保持  

---

### 🔹 2. イベント登録機能
- カテゴリー選択（ライブ / 握手会 / 配信 / ライブ応募 etc）  
- 選択したカテゴリーに応じて入力項目を自動切替  
- 確認画面 → 登録完了 → 詳細画面へ遷移  
- 入力内容はセッションで保持  

---

### 🔹 3. イベント詳細ページ
- カテゴリーに応じて表示項目を出し分け  
- 推し名をクリック → 推し別のイベント一覧へ遷移  
- 編集ボタン/削除ボタンを配置  

---

### 🔹 4. 編集・削除機能
- 編集画面 → 確認画面 → 更新  
- 削除時は確認アラートを表示  
- 削除後はトップページへリダイレクト  

---

### 🔹 5. 推し一覧
- 推し名を「グループ」または「個人」で登録  
- イベント登録時にプルダウンで選択可能  
- 一覧から推し別イベントを確認可能  

---

### 🔹 6. セッション利用
- カレンダー表示月の保持  
- 登録／編集フォームの入力保持  
- カテゴリー選択状態の保持  
- 検索条件保持（実装予定）
